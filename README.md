# 江西农业大学教务系统抢课脚本

自动化抢课脚本，基于 Python + requests，支持验证码自动识别、Session 保持、轮询监控。

## 功能

- 自动登录江西农业大学教务系统（ddddocr 离线 OCR 识别验证码）
- 登录后自动扒取全校公选课列表，表格展示
- 交互式选择目标课程（支持序号 / 编号 / 关键词）
- 单线程轮询监控课程余量，有余量立即提交选课
- 课程已满时支持切换目标或继续等待
- Session 过期自动重登录

## 环境要求

- Python 3.8+
- Windows / macOS / Linux

## 安装

```bash
git clone https://github.com/xlx0131/-JXAU-.git
cd -JXAU-
pip install -r requirements.txt
```

或手动安装依赖：

```bash
pip install requests beautifulsoup4 lxml ddddocr
```

## 使用

```bash
python jxau_course_grabber.py
```

按提示输入学号、密码，学年学期直接回车使用默认值（自动推断当前学年度）。

登录后自动展示可选课程列表：

```
========================================================================================================================
 序号   教学班编号       课程名称                             教师        学分   已选/容量   状态   上课时间
========================================================================================================================
    1   20261314001    宝石鉴定与欣赏4001班                 章俊霞         1      62/60      满    星期一 晚上 9-11节
    2   20261317001    [人文]:世界战争概要7001班             刘后根         1     120/120     满    星期一 晚上 9-11节
  ...
========================================================================================================================
```

输入目标课程（序号 / JxbBh / 关键词）：

```
请选择目标课程 > 20261317001
```

选课后进入抢课循环，有余量时自动提交。按 `Ctrl+C` 停止。
