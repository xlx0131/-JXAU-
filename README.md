# 江西农业大学教务系统自动化工具集

一套基于 Python 的江西农业大学教务系统自动化工具，涵盖**自动抢课**、**成绩查询**与**课表查询**三大场景。利用 `requests` 库模拟自研 ASP.NET MVC 教务系统 HTTP 请求，配合 `ddddocr` 离线 OCR 实现验证码自动识别，无需人工干预。

> **🚀 新增：GUI 一体化桌面应用！** 三个脚本整合到统一图形界面，三标签页切换 + 共享登录会话，无需分别运行各脚本。前往 [工具四：GUI 一体化桌面应用](#工具四gui-一体化桌面应用-jxau_jiaowu_gui) 下载 `JXAU教务一体化.exe`。

---

## 工具一：抢课脚本 (`jxau_course_grabber.py`)

自动化抢课脚本，支持定时轮询、验证码自动识别、交互式课程选择。

### 功能特性

- **验证码自动识别** — 基于 ddddocr 离线 OCR，自动识别登录验证码；识别失败自动降级手动输入
- **课程浏览与筛选** — 登录后自动拉取全校公选课列表，以规整表格展示（序号 / JxbBh / 课程名 / 教师 / 学分 / 余量）
- **交互式选课** — 支持按序号、JxbBh 编号、课程名称关键词选择目标课程
- **定时抢课** — 单线程轮询 + 随机延时（0.5~1.5s），自动检测课程余量，有余量立即提交
- **登录重试机制** — 登录失败（验证码错误/网络波动）时自动重试，支持最大重试次数与重试间隔配置
- **会话自动维护** — Session 过期自动检测并重新登录，支持持续运行
- **结构化日志** — 同时输出控制台和文件，方便排查问题

### 使用方式

```bash
python jxau_course_grabber.py
```

按提示输入学号、密码，学年学期回车取默认值（自动推断当前学期）。登录后将展示可选课程列表，输入目标课程后进入抢课循环，按 `Ctrl+C` 停止。

**环境变量配置（非交互模式）：**

```bash
set JXAU_STU_ID=2024xxxxxx
set JXAU_PASS=your_password
python jxau_course_grabber.py
```

## 工具二：成绩查询脚本 (`jxau_grade_query.py`)

快速查询全部学期成绩并导出，支持按学期分组展示与 CSV 导出。

### 功能特性

- **全学期成绩拉取** — 一键拉取教务系统中全部已有成绩数据
- **按学期分组展示** — 每学期独立表格，自动汇总学期学分、及格/不及格统计
- **详细统计汇总** — 总课程数、总学分、平均绩点（仅统计有效绩点课程）
- **CSV 导出** — 支持将全部成绩导出为 UTF-8 BOM 编码的 CSV 文件，可用 Excel 直接打开
- **验证码自动识别** — 同抢课脚本一致的 ddddocr 离线识别方案

### 使用方式

```bash
python jxau_grade_query.py
```

按提示输入学号、密码。登录后自动拉取并展示全部成绩，程序将询问是否导出 CSV。

**环境变量配置（非交互模式）：**

```bash
set JXAU_STU_ID=2024xxxxxx
set JXAU_PASS=your_password
python jxau_grade_query.py
```

## 工具三：课表查询脚本 (`jxau_schedule_query.py`)

查询本人课表，支持按学期切换、网格化课表展示（上午 / 下午 / 晚上分区）。

### 功能特性

- **登录后自动加载课表** — 登录教务系统后自动获取学期列表，默认加载最新学期课表
- **网格化课表展示** — 按星期 X 节次组织，上午（1-2 节）、下午（5-6 节）、晚上（9-11 节）清晰分区
- **课程详情** — 每门课程展示课程名称、授课周次、任课教师、教室、教室类型
- **学期切换** — 支持交互式切换学期，查看过往或未来学期课表
- **验证码自动识别** — 同系列一致的 ddddocr 离线识别方案

### 使用方式

```bash
python jxau_schedule_query.py
```

按提示输入学号、密码。登录后自动展示当前学期课表，输入学期编号可切换学期查看。

**环境变量配置（非交互模式）：**

```bash
set JXAU_STU_ID=2024xxxxxx
set JXAU_PASS=your_password
python jxau_schedule_query.py
```

## 工具四：GUI 一体化桌面应用 (`JXAU教务一体化.exe`)

将上述三个脚本整合为统一的 Windows 桌面应用程序，免安装 Python 环境，双击即用。

### 功能特性

- **三标签页界面** — 抢课选课、成绩查询、课表查询三个模块集成在一个窗口
- **共享登录会话** — 登录一次后三个模块共用 Session，无需重复输入账号密码
- **可视化课表** — Canvas 网格化渲染，上午/下午/晚上分区，支持垂直滚动
- **成绩 Treeview** — 表格化成绩明细，学期切换 + 汇总统计
- **实时运行日志** — 底部日志面板，所有操作可见、可追踪
- **便携免安装** — 单文件 exe，拷贝即用，无需 Python 环境

### 下载与使用

从 [Releases](https://github.com/xlx0131/-JXAU-/raw/main/JXAU%E6%95%99%E5%8A%A1%E4%B8%80%E4%BD%93%E5%8C%96.exe) 下载 `JXAU教务一体化.exe`，双击运行即可。

> **注意**：首次运行时 Windows Defender 可能弹出 SmartScreen 警告，点击「更多信息」→「仍要运行」即可。这是因为 exe 未经过数字签名，属正常现象。

## 环境依赖

- Python 3.8+
- `requests` — HTTP 请求库
- `beautifulsoup4` — HTML 解析（仅抢课脚本需要）
- `lxml` — 底层 HTML/XML 解析器（仅抢课脚本需要）
- `ddddocr` — 离线验证码 OCR 识别

### 安装

```bash
pip install requests beautifulsoup4 lxml ddddocr
```

或使用项目提供的 `requirements.txt`：

```bash
pip install -r requirements.txt
```
